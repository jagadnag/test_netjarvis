
**SSH Configuration** (High)
```
crypto key generate rsa modulus 2048
ip ssh version 2
ip ssh time-out 60
ip ssh authentication-retries 3
```

**VTY Line Security** (High)
```

```

**HTTP/HTTPS Server** (High)
```
no ip http server
ip http secure-server
```

**Console Security** (Medium)

**IP Source Routing** (Medium)
